# Student Life CLI Tracker

This is my first beginner Python project.

## Project Goal

Build a command-line app that helps track college tasks, coding goals, and weekly progress.

## Version 1.0 Features

* Show a menu
* Add a task
* View all tasks
* Mark a task complete
* Delete a task
* Exit the program
* Handle basic invalid menu choices
* Handle basic invalid task choices
