# Project 1: Student Life CLI Tracker

## Purpose

This will be a beginner Python command-line app that helps me track college tasks, coding goals, and weekly progress.

I chose this project because it connects to my real life as an incoming college student and helps me practice Python while building something useful.

## Version 1 Features

- Add a task
- View all tasks
- Mark a task complete
- Delete a task
- Track weekly coding goals

## Future Features

- Save tasks to a file
- Add task categories
- Add due dates
- Add weekly progress summaries
- Turn it into a web app later

## Skills This Project Will Help Me Practice

- Python input and output
- Variables
- Lists
- Dictionaries
- If/else statements
- Loops
- Functions
- Basic file saving later
- Project organization
